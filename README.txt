# Macrocosmic
version of script:
    - index.js: known stable version
    - beta.js: testing purposes, may be unstable
server instructions:
    first, you will need to install node.js @ https://nodejs.org/en/download/ (may not need to install chocolate)
    second, it is recommended to use visual studio [[ code ]] for easier editing of source.
    third, install all dependencies for the server to work. this includes https://www.npmjs.com/package/bytebuffer and https://www.npmjs.com/package/ws
        node i bytebuffer
        node i ws
    lastly, run the server/index.js file.
        node .
